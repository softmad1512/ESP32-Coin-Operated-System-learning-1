# ESP32 Coin Operated System

##  專案介紹
本專案使用 ESP32、投幣器、繼電器、OLED顯示器
實現室內遊樂場遊戲機投幣控制系統
---
## 使用功能
- 投幣計數（Interrupt）
- 防抖動（Debounce）
- 累計投幣啟動繼電器
- OLED 顯示幣數
---
##  技術重點
- ISR（中斷處理）
- Finite State Machine（狀態機）
- Non-blocking 設計（millis）
- Embedded System Design
---
##  硬體
- ESP32
- Coin Acceptor
- Relay Module
- SSD1306 OLED
